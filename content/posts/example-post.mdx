# MDX 포스팅 시스템

이것은 **next-mdx-remote-client**를 사용한 MDX 포스팅 시스템의 예시입니다.

## 주요 기능

이 시스템의 주요 기능들을 소개합니다:

### 1. 문단별 댓글

각 문단마다 댓글을 달 수 있습니다. 이 문단에도 댓글을 달아보세요!

### 2. 마크다운 지원

일반적인 마크다운 문법을 모두 지원합니다:

- **굵은 텍스트**
- _기울임 텍스트_
- `인라인 코드`

### 3. 코드 블록

```javascript
const greeting = (name) => {
  return `안녕하세요, ${name}님!`;
};

console.log(greeting('개발자'));
```

### 4. 인용구

> 이것은 인용구입니다. 중요한 내용을 강조할 때 사용합니다.

### 5. 인터랙티브 컴포넌트

MDX에서는 React 컴포넌트도 사용할 수 있습니다. 아래는 카운터 버튼 예시입니다:

## 댓글 기능 테스트

이 문단에서 댓글 기능을 테스트해보세요. 각 문단마다 독립적으로 댓글을 관리할 수 있습니다.

다음 문단도 별도의 댓글 영역을 가집니다.

## 마무리

MDX를 활용하면 리치한 컨텐츠를 쉽게 작성할 수 있습니다. 문단별 댓글 기능으로 독자와의 소통도 원활하게 할 수 있습니다.
