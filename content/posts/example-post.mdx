## 문단 및 텍스트 스타일

이것은 일반 문단입니다. **커스텀 Paragraph 컴포넌트**가 사용되며 클릭하면 문단별 댓글 기능이 활성화됩니다.

**굵은 텍스트**, _기울임 텍스트_, 그리고 `인라인 코드`를 포함한 다양한 텍스트 스타일을 지원합니다.

## 코드 블록 테스트

```javascript
// 커스텀 CodeBlock 컴포넌트
const greeting = (name) => {
  return `안녕하세요, ${name}님!`;
};

console.log(greeting('개발자'));
```

```python
# Python 코드 예시
def hello_world():
    print("Hello, World!")
    
hello_world()
```

```css
/* CSS 코드 예시 */
.custom-style {
  background-color: #f0f0f0;
  border-radius: 8px;
  padding: 1rem;
}
```

## 링크 테스트

[내부 링크 예시](/posts) 및 [외부 링크 예시](https://github.com)를 테스트합니다. 커스텀 Link 컴포넌트가 적용되어 외부 링크는 새 탭에서 열립니다.

## 인용구 테스트

> 이것은 커스텀 Blockquote 컴포넌트입니다. 중요한 내용을 강조할 때 사용하며, 좌측에 테두리와 함께 이탤릭 스타일이 적용됩니다.

## 목록 테스트

### 순서 없는 목록 (커스텀 UnorderedList)

- 첫 번째 항목
- 두 번째 항목
  - 중첩된 항목 1
  - 중첩된 항목 2
- 세 번째 항목

### 순서 있는 목록 (커스텀 OrderedList)

1. 첫 번째 단계
2. 두 번째 단계
3. 세 번째 단계
   1. 세부 단계 1
   2. 세부 단계 2

## 테이블 테스트

| 헤더 1 | 헤더 2 | 헤더 3 |
|--------|--------|--------|
| 데이터 1 | 데이터 2 | 데이터 3 |
| 긴 데이터 예시 | 중간 데이터 | 짧은 데이터 |
| 테스트 | 커스텀 테이블 | 컴포넌트 |

위 테이블은 커스텀 Table, TableHead, TableBody, TableRow, TableHeaderCell, TableDataCell 컴포넌트를 사용합니다.

## 모든 컴포넌트 정리

이 문서에서 테스트된 커스텀 컴포넌트들:

1. **제목 컴포넌트**: H1, H2, H3, H4, H5, H6
2. **텍스트 컴포넌트**: Paragraph (문단별 댓글 기능), InlineCode
3. **코드 컴포넌트**: CodeBlock (언어별 syntax highlighting)
4. **링크 컴포넌트**: Link (내부/외부 링크 구분)
5. **인용구 컴포넌트**: Blockquote
6. **목록 컴포넌트**: UnorderedList, OrderedList, ListItem
7. **테이블 컴포넌트**: Table, TableHead, TableBody, TableRow, TableHeaderCell, TableDataCell
8. **기타 컴포넌트**: Image, HorizontalRule
9. **인터랙티브 컴포넌트**: CounterButton

모든 컴포넌트가 정상적으로 렌더링되고 스타일이 적용되는지 확인해보세요!
